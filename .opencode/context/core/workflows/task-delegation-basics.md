# Task Delegation Basics

- **Delegate** when a task touches > 3 files, requires specialized knowledge, or has complex dependencies.
- **Execute directly** for single‑file, straightforward changes.
- When delegating, create a **context bundle** that includes:
  - Relevant standards files (the ones you just added).
  - The task description and acceptance criteria.
  - Any existing code snippets needed for reference.
- Sub‑agents should read the bundle before starting work.
